Tulu Font

Darage v1.1 Traditional Tulu Font

Product Details

  • Script Tulu
  • Languages Tulu, Sanskrit
  • License SIL OpenSource
  • Extension OpenType
  • Version 1.1



Darage v1.1 is an independent traditional calligraphy style font for Tulu developed by Prahlad Tantry. The style of this font presents a solid yet vivid personality and fosters the traditional values of the Tulu script. Dareg compounds a wide number of conjunct letters which makes this particular font stand out from the rest.

The font is free to use and share, and is licenced under SIL OpenSource (OFL) terms.
