Licensing

Types of Licenses

IDCL provides two font licenses:


  • SIL Open Font License(OFL)
  • IDCL Free Font License (IFFL)
SIL Open Font License

The SIL Open Font License (OFL) is a free, libre and open source license specifically designed for fonts and related software. It allows the licensed fonts to be used, studied, modified and redistributed freely as long as they are not sold by themselves.

The SIL OFL is intended to be used with fonts that are either entirely original works, or modifications of other fonts that are in the public domain or are distributed under a license that is compatible with the SIL OFL.

The SIL OFL has several key provisions that make it a flexible and permissive license for fonts:

  • It allows the licensed fonts to be used for any purpose, including commercial use.
  • It allows the licensed fonts to be modified and redistributed.
  • It allows the licensed fonts to be bundled with other software, such as operating systems or word processors.
  • It requires that the original copyright notice and license information be included with any copies of the licensed fonts.
  • It requires that modified versions of the licensed fonts be distributed under a different name, to prevent confusion with the original version.
IDCL Free Font License

IDCL Free Font License (IFFL) is a kind of Font License used by IDCL that permits fonts to be freely downloaded, copied, provided with any software, and to be shared provided that the user abide to the terms and conditions of the License.

While the IFFL appears to be similar to OFL, it does impose some restrictions on the user:

  • Fonts with IFFL Licensing cannot be modified in any manner without the explicit written permission of the designer.
  • It may be bundled with any software/package, and even if such software costs money, the distributer cannot charge any cost for the Font.

Installing Fonts

Download and Unpack

After Downloading the Font which will be in ZIP Archive (.zip) from IDCL website, Unpack it using an archiving software such as 7zip, Winrar, Archive Manager, Z-archiver (Android).

In Windows XP or higher, you can double-click the .zip file to extract it.

In most Linux Distributions, you can right click on the .zip file and extract it. If your Linux Distribution does not have an archiving software, you can install Peazip, File Roller, Engrampa, 7-zip, tar, Ark etc.

In Mac OS X, double-click and extract the files.

In Android, Files by Google allows you to extract a .zip file.

  • Navigate to your Downloads Folder using Files by Google App
  • Tap the file.
  • Tap Extract to unzip the file. After extractio, you will see a list of unzipped files.

Alternatively you can use Z-archiver in Android.
Installing on Windows

On Windows 7 and Higher

  • Select the font files (.ttf)/(.otf)
  • Right click and select install

    • Alternatively,
  • Go to Start> Control Panel> Appearance and Personalisation > Fonts
  • Click on File> Install New Fonts
  • Go to the location where you have extracted the font files in the Add Fonts dialog box
  • Select the Font files and click Ok.
Installing on Linux

On Ubuntu & Ubuntu based distributions

  • Double click the (.ttf)/(.otf)
  • It will open 'Font Viewer' application.
  • Click on 'Install' in the top-right corner.

    • Alternatively
  • Open terminal and copy paste this command sudo apt install font-manager
  • After Installation, open Font Manager from the App Menu
  • Click the '+' button on the top-left corner and choose the font files to install.
Installing on Mac
  • Double click the (.ttf)/(.otf) file. This will open a preview window.
  • Click the 'Install Font' button

Typing Help

Typing Indian Languages in Windows, Linux and Mac is easy now. However, the issue is that only major language scripts are available as Keyboard layouts. What if you want to type Brahmi or Mahajani? You can install utilities such as Keyman or build your own layouts using any Keyboard Layout editor software. IDCL is constantly working on the development of Keyman Keyboard layouts for Windows, Linux, Mac, Android and Iphone. Watch out for the announcements.


Typing in Windows

In windows you can install phonetic keyboard for Hindi, Bangla, Tamil, Marathi, Punjabi, Gujarathi, Odiya, Telugu, Kannada and Malayalam. Indic Traditional INSCRIPT keyboards are also available in Windows.


  • Go to Time & Language in Settings and select the language from the navigation menu
  • Click on the + icon, and add the preferred language
  • Go to language page, select the language and then click the options button to go to language options page.
  • Add the keyboard by clicking the + icon and then select the type of keyboard
  • Enable the phonetic keyboard by clicking on the input indicator on the taskbar

    • For more details visit Microsoft help
Typing in Linux

For Ubuntu based distributions


  • Open terminal, copy and paste sudo apt install ibus-m17n
  • Open 'Startup Applications' and click Add
  • Give name as 'ibus', and command as 'ibus-daemon' and click Add.
  • Got Settings>Region & Language
  • Under Input Sources, click the '+' button.
  • Click on the three vertical dots.
  • Search for the language you need
  • Choose a Keyboard layout. (hovering the mouse displays a preview)
  • Finally click Add.

  • On your toolbar, you can see 'en' (this is the keyboard selection). Click on it
  • Select the layout you want. Alternatively, you can configure a shortcut such as Ctrl+Space to switch between Keyboard layouts easily.

Get Custom fonts

IDCL makes custom fonts for specific usages such as research.

Request a Custom font for free

If you would like to see a historic or lesser known script with or without unicode support as a font, you can send a request to us via the contact form or by sending us an email. Please read the following guidelines.


  • Submit a request with your name (anonymous mail and nicknames are not accepted).
  • Submit the full details of the writing script along with research materials and photos of written samples (you can also send photos of your handwriting). Make sure that the letters are legible.
  • Submit the photo of the complete alphabet chart. The letters should be marked with either ISO or Phonetic script.
  • Enter additional details such as reference links.
  • Describe the purpose of a custom font for the script.


    • We will reply back to you within one week time informing you about the acceptance or rejection of your proposal.
    See this sample request.
