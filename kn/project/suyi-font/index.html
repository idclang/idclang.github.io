ಸುಯಿ ತುಳು

ಯೋಜನೆಯ ವಿವರಗಳು

  • ಲಿಪಿ : ತುಳು ಲಿಪಿ
  • ಭಾಷೆಗಳು : ತುಳು, ಸಂಸ್ಕೃತ
  • ಪರವಾನಗಿ : ವಾಣಿಜ್ಯ ಬಳಕೆಗೆ ಉಚಿತ
  • ಪ್ರಗತಿ : ವಿರಾಮಗೊಳಿಸಲಾಗಿದೆ
  • ಬಿಡುಗಡೆ ದಿನಾಂಕ : ತೀರ್ಮಾನವಾಗಿಲ್ಲ

ತುಳುವಿಗೆ ಆಧುನಿಕ ಫಾಂಟ್ ಐಡಿಸಿಎಲ್‌ನ ಮೊದಲ ಯೋಜನೆಯಾಗಿದೆ. ತುಲು ಭಾಷೆಗೆ ಸಾಂಪ್ರದಾಯಿಕ ಶೈಲಿಯ ಫಾಂಟ್ ಅಭಿವೃದ್ಧಿಪಡಿಸಲು ನಮ್ಮ ತಂಡ ನಿರ್ಧರಿಸಿದೆ.ತುಳು ಫಾಂಟ್ ಅಭಿವೃದ್ಧಿಪಡಿಸುವ ಮುಖ್ಯ ಸವಾಲು ಯುನಿಕೋಡ್ ಬೆಂಬಲದ ಅಲಭ್ಯತೆ. ಈ ಸ್ಕ್ರಿಪ್ಟ್ ಅದರ ಸಹೋದರಿ ಸ್ಕ್ರಿಪ್ಟ್ ಮಲಯಾಳಂನಂತೆ ಸಂಕೀರ್ಣವಾದ ಕಾರಣ ಈ ಬೆಂಬಲ ತುಂಬಾ ಅಗತ್ಯವಿದೆ.

ಮಲಯಾಳಂನ ವಿನ್ಯಾಸವನ್ನು ಆಧರಿಸಿ ತುಲುಗಾಗಿ ಅನೇಕ ಫಾಂಟ್‌ಗಳನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ, ಆದರೆ ಹೊಸ ವಿನ್ಯಾಸವನ್ನು ಟೈಪ್ ಮಾಡಲು ಕಲಿಯುವಲ್ಲಿನ ತೊಂದರೆ ಯಾವಾಗಲೂ ಬಳಕೆದಾರರನ್ನು ದೂರವಿರಿಸುತ್ತದೆ. ಆದಾಗ್ಯೂ, ಲ್ಯಾಟಿನ್ ವಿನ್ಯಾಸವು ಸಾಂಪ್ರದಾಯಿಕ ತುಳು ಫಾಂಟ್‌ಗೆ ಸೂಕ್ತವಲ್ಲ ಏಕೆಂದರೆ ಅದು ಅನೇಕ ಸಂಭಾವ್ಯ ಸಂಯೋಜಿತ ಅಕ್ಷರಗಳನ್ನು ಹೊಂದಿದೆ. ಮಲಯಾಳಂ ವಿನ್ಯಾಸದ ಮೇಲೆ ನಿರ್ಮಿಸುವುದು ಮತ್ತು ಅಂತಿಮ ಬಳಕೆದಾರರಿಗಾಗಿ ಲಿಪ್ಯಂತರ ಕೀಬೋರ್ಡ್ ಅನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸುವುದು ಉತ್ತಮ ಪರಿಹಾರವಾಗಿದೆ.
