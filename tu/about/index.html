How can you help?

This Initiative requires the help of volunteers. You can take part in this initiative if you;

  • speak any lesser-known or endangered Indian language.
  • are from any underresourced linguistic community.
  • can help by collecting data, prepare corpus data and provide details about underresourced languages.
  • know about historical scripts/literature or have access to such.
  • know computational linguistics.
  • can help create digital materials or tools for language development.
  • know graphic designing or typography.
  • can collect data about any non-standatd dialect
  • can connect with any underprivileged linguistic community.

see the documentation for more details
