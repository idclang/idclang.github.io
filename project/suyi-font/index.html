A Sans Font For Tulu

Suyi sans font for Tulu

Project Details

  • Script : Tulu Script
  • Languages : Tulu, Sanskrit
  • License : Free for commercial use
  • Progress : Paused
  • Release Date : pending

A modern font for Tulu was one of the first projects of IDCL. And the team decided to develop a traditional font for the Tulu language. The Main challenge of developing a Tulu font is the unavailability of Unicode support, and this support is very much needed because this script is complex like its sister script Malayalam.

Many fonts have been developed for Tulu based on the layout of Malayalam, but the difficulty in learning to type in a new layout has always kept the users away. However, the Latin layout is not apt for a traditional Tulu Font as it has many possible conjunct characters. The best solution is to build on top of a Malayalam layout and develop a transliteration keyboard for the end user.

Currently, the work is been haulted.
